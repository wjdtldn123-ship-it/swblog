소프트웨어 공학과 첫 전공 수업 후기

Hello, C World!

드디어 본격적인 전공 공부가 시작되었다. 첫 언어는 컴퓨터 공학의 근본이라는 C언어! 교수님께서 화면에 띄워주신 예제 코드를 따라 쳤을 때, 터미널에 내가 작성한 글자가 그대로 출력된 것을 보고 묘한 감동을 느꼈다.

#include <stdio.h>

int main() {
    printf("Hello, CS World!\n");
    return 0;
}

어려웠던 점과 깨달은 점

처음에는 `include`가 뭔지, 왜 `main` 함수가 있어야 하고 `return 0;`을 써야 하는지 모든 게 낯설었다. 하지만 예습과 복습을 통해 점차 컴퓨터가 코드를 읽고 실행하는 흐름을 이해하게 되었다.

앞으로 데이터 타입, 반복문 제어문, 포인터와 구조체 같은 어려운 산이 쭉 기다리고 있다지만 차근차근 부딪혀볼 생각이다. 오늘 첫 코드 작성을 성공적으로 해낸 기억을 잊지 말고, 졸업하는 그날까지 파이팅해야겠다!
