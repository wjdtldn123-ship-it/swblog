소프트웨어 공학과
첫 전공 수업 후기

기대 반 걱정 반으로 시작한 첫 전공 수업, C언어 프로그래밍.

1. 시작하며

드디어 기다리고 기다리던 전공 과목의 첫 수업이 시작되었습니다. 고등학교 때 취미로 파이썬을 아주 조금 만져본 적은 있지만, C언어 같은 본격적인 로우레벨(Low-level) 언어는 처음 접해보아서 매우 떨렸습니다.

hello.c 
#include <stdio.h>

int main() {
    printf("Hello World!\n");
    return 0;
}

2. 앞으로의 다짐

앞으로 매주 배운 내용을 이 블로그에 정리할 계획입니다. C언어뿐만 아니라 독학으로 진행하고 있는 웹 개발, 알고리즘 문제 풀이도 꾸준히 올려보려 합니다.
